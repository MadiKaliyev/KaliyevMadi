Linear Equtions
A linear equation is any equation that can be written in the form
ax + b = 0
Where a and b are real numbers and x is a variable. This form is sometimes called the standard form of a linear
equation. Note that most linear equations will not start off in this form. Also, the variable may or may not be an x there.

Quadratic Equation
The standard form of a quadratic looks like this:
ax2 + bx + c = 0
Where a, b, c are known values and x is the variable or unknown (we don`t know it yet). Also, a can`t be 0

The area of a circle (A)
π (Pi) lines the radius (r) squared.
A = π × r2
or, when you know the diameter (D):
A = (π / 4) × D2
or, when you know the circumference (C):
A = C2 / 4 × π
Intersection
The intersection of two sets A and B, denoted by A ∩ B, is the set of all objects that are members of both the sets
A = ∩ B = [ x : x ∈ A and x ∈ B]
That is, x is an element of the intersection A ∩ B if and only if x is both an element ofA and an element of B
